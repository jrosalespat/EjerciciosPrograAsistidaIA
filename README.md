# PrograAsistidaIA

## Descripción
Conjunto de ejercicios en JavaScript para practicar y reforzar conceptos de programación.

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/PrograAsistidaIA.git

# Navegar al directorio
cd PrograAsistidaIA
```

## Uso

```bash
# Ejecutar los ejercicios con Node.js
node ejercicio1.js
```

## Estructura del Proyecto
- `ejercicio1.js` - Primer ejercicio
- `ejercicio2.js` - Segundo ejercicio
- ... (más ejercicios)

## Características
- Ejercicios variados de JavaScript
- Código comentado y documentado
- Práctica de conceptos fundamentales
